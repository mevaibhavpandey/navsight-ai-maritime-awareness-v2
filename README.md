# 🚢 NavSight AI — Maritime Situational Awareness Platform

NavSight AI is an advanced web-based maritime intelligence and monitoring system designed to enhance maritime domain awareness through real-time vessel tracking, geospatial visualization, OCR-based data ingestion, and automated alert generation.

The platform provides a command-center style interface for monitoring maritime activity, analyzing vessel behavior, and managing maritime security zones.

---

## 🌍 Key Features

### 🗺️ Interactive Maritime Mapping

* Satellite map visualization using Leaflet
* Indian 200-NM Maritime Boundary display
* Patrol zones, piracy zones, exercise areas, and checkpoints
* Layer toggling with real-time updates

### 🚢 Vessel Tracking & Intelligence

* Manual vessel entry and automated data ingestion
* Friendly vs suspicious vessel classification
* Flag detection and registry parsing
* Vessel trajectory prediction

### 📂 Multi-Format Data Processing

Supports:

* TXT / MD
* DOCX
* XLSX / CSV
* JSON
* Images (JPG, PNG, WEBP, BMP)

### 🔍 OCR Integration

* Offline OCR using Tesseract.js
* Automatic extraction of vessel information from images

### ⚠️ Smart Alert System

* Maritime boundary violation alerts
* Suspicious vessel detection
* Priority-based alert classification

### 💾 Offline-First Architecture

* IndexedDB storage
* Data import/export
* Upload history tracking

### 📊 Command Dashboard

* KPI monitoring
* Vessel statistics
* Alert overview
* System health indicators
